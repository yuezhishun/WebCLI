# WebCLI Vue Frontend

这是一个使用 Vue 3 + Vite 构建的 WebCLI 前端。

## 安装依赖

```bash
npm install
```

## 开发模式

```bash
npm run dev
```

开发服务器将在 http://localhost:3000 启动。

## 构建生产版本

```bash
npm run build
```

## 预览生产版本

```bash
npm run preview
```

## 功能特性

- 创建终端实例
- 连接到终端实例
- 终端交互
- 加载历史记录
- 重新同步会话
- 断开连接
- 终止终端实例

## 技术栈

- Vue 3 (Composition API)
- Vite
- @xterm/xterm
- @xterm/addon-fit
